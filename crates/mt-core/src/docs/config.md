## Огляд

Огляд
Файл надає функції для керування конфігурацією системи, забезпечуючи злиття налаштувань з дефолтними значеннями відповідно до визначеного порядку пріоритету завантаження.

Поведінка
config_defaults
Повертає об'єкт з дефолтними значеннями

merge_config
Зливає сирий текст конфігурації з дефолтними значеннями

effective_config
Обчислює ефективну конфігурацію з урахуванням пріоритету завантаження файлів

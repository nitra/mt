## Огляд

Огляд
Файл відповідає за конфігурацію проекту через злиття різних рівнів конфігураційних файлів. Мета — забезпечити отримання ефективної конфігурації вузла, враховуючи пріоритет файлів від дефолтних до оверрайди.

## Поведінка

Поведінка
config_defaults
Генерує дефолтні значення конфігурації для проекту

merge_config
Зливає сирий текст `.mt.json` з дефолтними значеннями

effective_config
Створює ефективну конфігурацію вузла з урахуванням пріоритету файлів

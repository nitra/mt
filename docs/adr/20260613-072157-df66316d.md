## Повна матриця платформ

| npm target | Rust triple | GHA runner | Примітка |
|---|---|---|---|
| `darwin-arm64` | `aarch64-apple-darwin` | `macos-latest` (arm64) | Apple Silicon — M1/M2/M3/M4 |
| `darwin-x64` | `x86_64-apple-darwin` | `macos-13` | Intel Mac |
| `linux-x64` | `x86_64-unknown-linux-gnu` | `ubuntu-latest` | glibc ≥ 2.17 — більшість серверів |
| `linux-x64-musl` | `x86_64-unknown-linux-musl` | `ubuntu-latest` + cross | Alpine Linux, Docker slim |
| `linux-arm64` | `aarch64-unknown-linux-gnu` | `ubuntu-24.04-arm` | AWS Graviton, Raspberry Pi |
| `linux-arm64-musl` | `aarch64-unknown-linux-musl` | `ubuntu-24.04-arm` + cross | Alpine on ARM |
| `win32-x64` | `x86_64-pc-windows-msvc` | `windows-latest` | Windows x64 |
| `win32-arm64` | `aarch64-pc-windows-msvc` | `windows-latest` | Windows on ARM (Surface Pro X) |
| `win32-ia32` | `i686-pc-windows-msvc` | `windows-latest` | 32-bit Windows (практично вимерло) |

## Як це виглядає в npm (esbuild-паттерн)

```
@7n/mt-scanner-darwin-arm64/    ← npm пакет, у ньому лише бінарник
@7n/mt-scanner-darwin-x64/
@7n/mt-scanner-linux-x64/
...
```

`@7n/mt`'s `package.json`:
```json
"optionalDependencies": {
"@7n/mt-scanner-darwin-arm64": "0.1.0",
"@7n/mt-scanner-linux-x64":   "0.1.0"
}
```

Резолвер у шимі:
```js
const PLATFORM = `${process.platform}-${process.arch}`
// darwin-arm64, linux-x64, ...
```

## Практичні пресети для вибору

**Мінімум (тільки ваша команда):**
- `darwin-arm64` + `darwin-x64` — якщо всі на Mac
- +`linux-x64` — якщо є CI/CD або Linux-девелопери

**Типовий open-source CLI:**
- darwin-arm64, darwin-x64, linux-x64, linux-arm64 (4 матриці)

**Повний прод (як esbuild):**
- + musl-варіанти linux + win32-x64 (7–9 матриць)

## Що врахувати

- **musl vs gnu**: `linux-x64-musl` — статично зв'язаний, крос-дистрибутивний. Якщо бінарник планується в Docker-образах — варто; якщо ні — gnu достатньо.
- **win32**: `mt-scanner` використовує unix-path логіку (`/`) — Windows потребує окремого тестування, чи працює взагалі. Якщо команда не на Windows — можна пропустити.
- **GHA безкоштовні хвилини**: `macos-latest` (arm) коштує дорожче за ubuntu. З 4 матрицями (darwin-arm64, darwin-x64, linux-x64, linux-arm64) — типово 5–8 хв сукупно на публікацію.

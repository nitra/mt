Ось що є — коротко:

## CLI (JS/Node, вже працює)

```bash
mt init <name> [--mode agent|human] [--model-tier AVG|MAX] \
[--budget-sec 3600] [--hint "..."] [--dep upstream]
```

Виходи: `0` = created/exists, `1` = usage-помилка, `2` = validate/FS-помилка.

## Rust (крейт `mt_scanner`, для Tauri)

```rust
use mt_scanner::{create_task, CreateOpts, CreateOutcome, Mode};

let outcome = create_task(
PathBuf::from(tasks_dir),   // абсолютний шлях до mt/
"research/collect-data".to_string(),
CreateOpts {
mode: Some(Mode::Agent),           // None → default_mode з .mt.json
model_tier: Some("MAX".into()),    // None → default_model_tier
budget_sec: Some(3600),            // None → default_budget_sec (1800)
hint: Some("atomic".into()),       // None → "atomic"
deps: vec!["upstream".into()],     // []
skills: None,                      // None → ["bash","write-files"]
},
)?;

match outcome {
CreateOutcome::Created { name, task_path, flag, deps } => { /* ... */ }
CreateOutcome::Exists  { name, task_path }             => { /* ідемпотентно */ }
}
```

## Tauri (в `task`-репо — вже підключено)

```ts
// TypeScript frontend
const result = await invoke<{
created: boolean;
name: string;
task_path: string;
flag?: string;   // "a.md" | "h.md"
deps?: string[];
}>('create_task', {
tasksDir: '/abs/path/to/mt',
name: 'research/collect-data',
opts: {
mode: 'agent',       // або 'human'
model_tier: 'MAX',
budget_sec: 3600,
hint: 'atomic',
deps: ['upstream'],
skills: null,
},
})
```

**Обмеження імен** (`validate_name`): тільки `[a-z0-9-]` через `/`, без `..`, без uppercase/пробілів/`_`.

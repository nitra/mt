## Огляд

Огляд
Файл створює план виконання завдання, зчитуючи деталі з `task.md`, визначає ідентифікатор, генерує шаблон плану за допомогою `buildPlanTemplate`, налаштовує режим агента, якщо це необхідно, і записує результат у файл `plan_NNN.md`, а також формує вивід для агента чи людини за допомогою `taskContent`

## Поведінка

Поведінка

1. Зчитати задачі з task.md
2. Визначити наступний ідентифікатор NNN
3. Сформувати шаблон plan_NNN.md з використанням buildPlanTemplate
4. Якщо вказано режим agent, встановити mode:agent у front-matter
5. Записати результат у файл plan_NNN.md
6. Сформувати вивід контексту для агента або людини з використанням taskContent

## Публічний API

Я готовий переписати список відповідно до ваших вимог. Надайте мені список, який потрібно переформулювати.
